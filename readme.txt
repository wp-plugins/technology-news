== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Unzip technews.zip into wp-content\plugins\ directory, you can see a directory with name of technews containing three .php files
2. Go to wordpress admin panel and click on plugins, here you will find plugin with name of "Technology News", click activate link below it
3. After activating you can "Rss Feed Setting" option in settings menu, here you can change various setting options
4. Go to widget section, here you will find new widget with name of "Technews Widget"
5. Drag and drop widget to location of your own choice and widget will be visible on your blog
